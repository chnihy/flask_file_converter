# flask_file_converter
Flask web app that converts file formats

Currently supports:
- YAML
- XML

## To install
1. Clone repo
'''
git clone https://github.com/chnihy/flask_file_converter.git
'''
2. Cd to repo
'''
cd flask_file_converter
'''
3. Install requirements
'''
pip install -r requirements.txt
'''
4. Launch
'''
python3 app.py
'''
